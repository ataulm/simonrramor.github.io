Notitia is an online, search platform for users to find teaching and learning 'opportunities' in their area.

This product was created during my time at General Assembly. The brief was to create a RESTful MEAN stack application in small development teams. Our team consisted of 3 members, working over 7 days, resulting in a presentation of our application.

Notitia allows user to search for and post teaching and learning 'opportunities' within a set distance radius. Users sign up to the platform, creating an online profile from scratch or using the Facebook Oauth to pre fill there information. After this, users have access to the full capabilities of the site, including reviewing other users and commenting on posts. Users can then start posting their own 'opportunities' to the main feed.

Users can post eather an offer to teach or a request to learn, these are then added to an information feed on the main page. Users can add a 'Title', 'About' and 'Suggested meeting point' for each post. Post are automatically geolocated to the users currently location however this can overridden by adding in a preferd meeting location.

When looking for an opportunity, users can search by keyword or use the filter to narrow down their search. Users can filter by date, post type and distance from current location.

Going forward we plan to utilise the Google places feature of the Google Maps API to add in suggested meeting places within the users travel radius.
